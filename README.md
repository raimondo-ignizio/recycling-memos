# Recycling Memos

A PHP and MySQL project on Start2Impact

# Description

This PHP application lets you take track of recycling collections by creating and storing memos in a database. 

# Usage

The memos are stored through a form that asks for:
- A specific day
- A range of time at which the collection will happen
- The type of waste that will be collected

For a single day it is possible to specify up to three differents hour ranges and waste types.

In the memos page you can:
- View every stored memos
- View the memos related to the collections happening this week
- View the memos related to the collections happening today
- Delete a single memo*
- Delete every memos in the database*

**An alert will be prompted to confirm this action.*
